# Pilares de Contenido

## 1. SEO Tecnico Shopify

El pilar central y principal. Todo lo que hace que una tienda Shopify sea visible para Google a nivel tecnico. Este es el foco principal del contenido y servicios.

### Subtemas
- **Indexacion y renderizado**: JavaScript rendering en Shopify, crawl budget, canonical tags, hreflang
- **Core Web Vitals**: Optimizacion de velocidad especifica para temas Shopify, Liquid optimization
- **Schema markup**: Product schema, BreadcrumbList, Organization, FAQ, Review
- **Sitemaps**: Sitemaps de Shopify, limitaciones, workarounds para tiendas grandes
- **Redirects y estructura de URLs**: Manejo de URLs de colecciones, productos discontinuados
- **Apps y rendimiento**: Impacto de apps de terceros en performance, como evaluarlas
- **Theme development SEO**: Liquid best practices, meta tags dinamicos, paginacion
- **Auditorias SEO**: Como auditar una tienda Shopify, herramientas, metodologias
- **SEO para ecommerce**: Keyword research para productos, optimizacion de fichas, contenido comercial

### Angulos de contenido
- Errores comunes al auditar tiendas Shopify
- Comparativas de temas Shopify desde perspectiva SEO
- Casos de estudio de mejoras tecnicas y aumento de trafico organico
- Tutoriales de implementacion paso a paso
- Mitos del SEO en Shopify
- Auditorias SEO: que hace una buena vs una mala auditoria
- Herramientas de SEO: Screaming Frog, Ahrefs, Semrush, Sitebulb
- Cambios en algoritmos de Google y como afectan a Shopify

---

## 2. Arquitectura de Catalogo

Como estructurar la informacion de una tienda para que Google y usuarios entiendan el catalogo. Parte integral del SEO tecnico.

### Subtemas
- **Estructura de categorias**: Jerarquia, profundidad vs amplitud, naming conventions
- **Facets y filtros**: Como manejar URLs de filtros, canonicalizacion, indexacion selectiva
- **Paginas de coleccion**: Optimizacion de landing pages de categorias
- **Navegacion interna**: Breadcrumbs, menus, links contextuales, hub pages
- **Internal linking**: Estrategias de enlazado para ecommerce
- **Catalogo grande**: Manejo de tiendas con 10K+ SKUs
- **Search interno**: Como el search de Shopify afecta SEO

### Angulos de contenido
- Auditorias de arquitectura antes/despues
- Frameworks para decidir estructura de categorias
- Errores de arquitectura que matan el SEO
- Casos de estudio de reestructuraciones
- Templates y checklists

---

## 3. Analitica y Medicion

Infraestructura de datos: tracking, medicion, reporteria. Soporte para medir el impacto del SEO tecnico.

### Subtemas
- **Google Tag Manager**: Implementaciones para trackear impacto SEO
- **GA4 para ecommerce**: Medir trafico organico, conversiones, comportamiento
- **Google Search Console**: Analisis de rendimiento SEO, queries, indexacion
- **Data quality**: Medir correctamente el trafico organico
- **Looker Studio**: Dashboards para reportar impacto SEO a clientes

### Angulos de contenido
- Como medir el impacto del SEO tecnico en trafico y revenue
- Configurar GA4 para trackear trafico organico correctamente
- Dashboards para reportar resultados SEO
- Errores comunes al medir trafico organico

---

## 4. AI Search / GEO

El pilar emergente. Como AI Search (ChatGPT, Perplexity, Google AI Overviews) afecta al SEO de ecommerce.

### Subtemas
- **AI Overviews**: Impacto en CTR, como optimizar para aparecer
- **ChatGPT y ecommerce**: Como GPT recomienda productos, como influenciarlo
- **Perplexity y busqueda**: Diferencias con Google, implicaciones para ecommerce
- **Generative Engine Optimization (GEO)**: Tecnicas para ser citado por IAs
- **Structured data para AI**: Como el schema markup alimenta a las IAs
- **Content para AI vs SEO**: Diferencias y similitudes
- **Futuro del search**: Predicciones realistas vs hype

### Angulos de contenido
- Analisis de como las IAs responden queries de ecommerce
- Experimentos: "Le pregunte a ChatGPT por X producto"
- Datos de trafico desde AI Overviews
- Skepticismo pragmatico: que funciona, que es hype
- Frameworks para pensar en AI Search

## Rotacion de pilares

### Sugerencia de frecuencia (LinkedIn)
- **SEO Tecnico Shopify**: 50% del contenido (pilar principal)
- **Arquitectura de Catalogo**: 25% (parte integral del SEO)
- **Analitica y Medicion**: 15% (soporte para medir impacto SEO)
- **AI Search / GEO**: 10% (piloto, creciendo gradualmente)

### Cross-pollination
Los pilares se cruzan naturalmente:
- Arquitectura + SEO Tecnico = "Como la estructura de tu coleccion afecta tu indexacion"
- SEO Tecnico + Analitica = "Como medir el impacto de cambios tecnicos en trafico organico"
- AI Search + SEO = "Como optimizar para AI Overviews y mantener visibilidad"

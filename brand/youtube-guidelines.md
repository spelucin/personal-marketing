## Thumbnails

### Reglas de composicion
- **Texto grande y legible**: Maximo 4-5 palabras
- **Contraste alto**: Texto sobre fondo con buen contraste
- **Rostro humano**: Si aparece Alex, usar expresion clara
- **Colores de marca**: Azules monocromaticos, sin saturar
- **No clutter**: Un elemento visual dominante, resto es apoyo

### Anatomia de thumbnail
```
┌──────────────────────────────────────────────────┐
│                                                  │
│  ┌────────────┐                                  │
│  │  FOTO DE   │   TITULO                         │
│  │  ALEX      │   GRANDE                         │
│  │  (opcional)│   2-4 PALABRAS                   │
│  │            │                                  │
│  └────────────┘                                  │
│                                                  │
│                              [badge/tag opcional]│
└──────────────────────────────────────────────────┘
```

### Tipografia en thumbnail
- Titulo: 64-80px, weight 700
- Subtitulo: 36-48px, weight 600
- Tag/badge: 20-24px, weight 500

### Estilos de thumbnail

#### 1. Tutorial / How-to
- Foto de Alex a la izquierda (1/3 del frame)
- Titulo grande a la derecha (2/3 del frame)
- Fondo: gradiente azul suave o dot grid
- Ejemplo: "ARREGLA tu SEO en Shopify"

#### 2. Analisis / Review
- Screenshot de herramienta o sitio web
- Titulo superpuesto con fondo semitransparente
- Badge con categoria: "AUDITORIA", "REVIEW"

#### 3. Opinion / Contrarian
- Foto de Alex con expresion pensativa o seria
- Titulo provocativo
- Fondo monocromatico azul

#### 4. Data / Metrica
- Grafico o numero grande
- Titulo explicativo corto
- Fondo limpio con dot grid

### Colores
- Fondo: `#f4fdff` a `#eaf6ff`
- Texto: `#1560bd` o blanco sobre fondo azul oscuro
- Acentos: `#0d4a9e`
- Evitar: colores calidos, saturacion alta

### Dimension
2560 x 1440 px (zona segura: 1546 x 423 px centrada para TV, 1280 x 360 para desktop)

### Contenido
- Nombre: "Alex Spelucin"

- Redes sociales
- Foto de Alex (opcional)
- Estetica: dot grid, gradiente azul, monocromatico

## Titulos de video

### Formato
- Maximo 60 caracteres para visibilidad completa
- Palabras clave al inicio
- Usar numeros cuando aplique: "5 errores..."
- Provocativo pero no clickbait

### Ejemplos
- "El error #1 de SEO en tiendas Shopify"
- "Por que tu GA4 miente (y como arreglarlo)"
- "Audite 47 tiendas Shopify: esto es lo que encontre"
- "Arquitectura de catalogo: la guia definitiva"

---

## Descripciones

### Estructura
1. **Primeras 2 lineas**: Hook + resumen (visible sin expandir)
2. **Timestamps**: Capitulos del video
3. **Links**: Recursos mencionados, herramientas, servicios
4. **Bio**: Descripcion corta de Alex
5. **Hashtags**: Maximo 3-5 al final

### Template
```
[Hook o resumen del video en 1-2 lineas]

[Link a recurso mencionado o CTA]

⏱️ Capitulos:
00:00 Intro
01:30 [Tema 1]
05:45 [Tema 2]
12:20 [Tema 3]

🔗 Recursos:
- [Recurso 1](url)
- [Recurso 2](url)

---

Soy Alex Spelucin, consultor independiente de SEO tecnico para ecommerce Shopify.

🌐 spelucin.pro
💼 linkedin.com/in/spelucin

#SEO #Shopify #SEOTecnico
```

## Formato de video

### Estilo
- Directo, sin relleno (igual que la voz de LinkedIn)
- Pantalla compartida para tutoriales (Screaming Frog, GA4, GTM)
- Talking head para opiniones y analisis
- Graficos y diagramas para conceptos tecnicos

### Duracion sugerida
- **Tutoriales**: 8-15 minutos
- **Analisis**: 10-20 minutos
- **Opinion**: 5-10 minutos
- **Caso de estudio**: 12-25 minutos

---

## Playlists sugeridas

1. **SEO Tecnico Shopify**: Tutoriales y guias tecnicas
2. **Arquitectura de Catalogo**: Estructura y navegacion
3. **Analitica y Medicion**: GTM, GA4, tracking
4. **Auditorias en vivo**: Casos reales (anonimizados)
5. **AI Search / GEO**: El futuro del search

---

## Frecuencia

- **1 video por semana** (calidad sobre cantidad)
- **Shorts**: 2-3 por semana (clips de videos largos o tips rapidos)

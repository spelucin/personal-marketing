## Carousels (Documentos PDF)

### Estructura estandar
- **Slide 1**: Hook + titulo + nombre/marca
- **Slides 2-N**: Contenido principal (1 idea por slide)
- **Slide final**: CTA + bio + contacto

### Reglas de composicion
- Maximo 10-12 slides (LinkedIn limita a 100 paginas PDF pero menos es mejor)
- Cada slide es independiente: debe funcionar solo
- Titulo grande y legible en mobile
- Mucho espacio en blanco, no llenar todo el slide
- Numerar slides (excepto el primero y ultimo)
- Watermark sutil con monograma en esquina

### Tipografia en carousel
- Titulo: 56-64px, weight 700
- Subtitulo: 32-36px, weight 500
- Body: 24-28px, weight 400
- Caption: 16-18px, weight 500, uppercase, tracking 0.18em
- Metricas: 72-96px, weight 300

### Colores
- Fondo: `#f4fdff` o `#eaf6ff`
- Texto principal: `#1560bd`
- Acentos: `#0d4a9e`
- Dot grid sutil de fondo

### Anatomia de slide
```
┌─────────────────────────────────┐
│  [eyebrow: categoria]           │
│                                 │
│  TITULO GRANDE                  │
│  subtitulo o descripcion        │
│                                 │
│  ┌─────────────────────────┐   │
│  │  contenido visual       │   │
│  │  (diagrama, metrica,    │   │
│  │   lista, codigo)        │   │
│  └─────────────────────────┘   │
│                                 │
│  02/08              [monograma] │
└─────────────────────────────────┘
```

## Banners (Imagenes de post)

### Tipos
1. **Announcement**: Lanzamiento de herramienta, nuevo servicio
2. **Quote**: Frase provocativa o insight
3. **Data**: Metrica o estadistica visual
4. **Tutorial**: Paso a paso visual
5. **Comparison**: Antes/despues, A vs B

### Reglas
- Headline legible en mobile (texto grande)
- Maximo 2-3 lineas de texto
- CTA visual si aplica
- Dot grid sutil de fondo
- Gradiente diagonal suave

### Anatomia de banner (1200x667)
```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│  [eyebrow]                                              │
│                                                         │
│  TITULO PRINCIPAL EN                                    │
│  MAXIMO 2-3 LINEAS                                      │
│                                                         │
│  Subtitulo o CTA                                        │
│                                                         │
│                                          [monograma]    │
└─────────────────────────────────────────────────────────┘
```

## Posts de texto (sin imagen)

### Estructura de copy
Ver `knowledge/voice-guide.md` para la voz completa.

### Formato
- Primer comentario: links, CTAs adicionales
- Hashtags: maximo 1-2, al final o en primer comentario
- Bio signature: solo en thought leadership posts
- Divider "--" antes de la signature

---

## Frecuencia y calendario

### Sugerencia
- **3-4 posts por semana** (calidad sobre cantidad)
- **1 carousel por semana** (contenido profundo)
- **2-3 posts de texto** (insights, opiniones, tips)
- **1 banner** cuando haya announcement o launch

### Mejores horarios (LATAM)
- Martes a jueves: 8-10am o 12-1pm
- Evitar fines de semana para contenido tecnico

## Hashtags LinkedIn

### Permitidos (usar con moderacion)
- #SEO
- #SEOTecnico
- #Shopify
- #Ecommerce
- #Analytics
- #GA4
- #MarketingDigital

### Regla
Maximo 2 hashtags por post. Preferir cero si el contenido es claro sin ellos.

## Engagement

### CTAs efectivos
- "Comenta X para..." (lead magnet)
- "Link en primer comentario" (para links externos)
- "Charlamos?" (soft CTA)
- "Cual es tu experiencia con...?" (engagement)

### Evitar
- Pedir likes o shares explicitamente
- "Etiqueta a alguien que..."
- Engagement bait sin sustancia
